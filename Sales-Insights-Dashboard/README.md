# 📊 Sales Insights Dashboard

## 🚀 Overview

This project analyzes sales performance using Power BI to generate actionable business insights.

## 🛠 Tools Used

* Power BI
* Power Query
* DAX
* Excel

## 📊 Features

* Sales trend analysis
* Top 10 products by sales
* Category-wise sales distribution
* Region-wise performance
* Profit analysis with conditional formatting

## 📷 Dashboard Preview

![Dashboard](images/dashboard.png)

## 💡 Key Insights

* Technology category generated highest profit
* Furniture showed lower profitability
* Certain regions contributed more to overall sales

## 🎯 Conclusion

This dashboard helps businesses make data-driven decisions.
